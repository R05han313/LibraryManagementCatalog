# 📚 Library Management Catalog

A modern Java Swing-based Library Management Catalog application developed to efficiently manage books, members, and loan records through an intuitive graphical user interface.

---

## 📌 Project Overview

Library Management Catalog is a desktop application built using Java Swing that streamlines library operations through an easy-to-use graphical interface. The application enables efficient management of books, members, and book loans while providing useful statistics and data import/export capabilities.

---

## ✨ Features

### 📖 Book Management
- Add new books
- Edit existing books
- Delete books
- Search books
- Track book availability

### 👥 Member Management
- Register new members
- Update member details
- Delete members
- Manage membership information

### 🔄 Loan Management
- Issue books
- Return books
- Track active loans
- Maintain loan history

### 📊 Dashboard
- Total books
- Total members
- Active loans
- Available books
- Library statistics

### 📁 Data Management
- Save library records
- Load existing data
- Import data from CSV
- Export data to CSV

### 🎨 User Interface
- Modern Java Swing GUI
- Dashboard panel
- Interactive tables
- Dialog-based forms
- Clean and responsive layout
